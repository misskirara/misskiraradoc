# 【カスタム絵文字を制作されている方へ】  
いつも、Misskiraraへのカスタム絵文字のご提供を頂きましてありがとうございます。

現在は[こちらのフォーム](https://docs.google.com/forms/d/e/1FAIpQLSd5lpr2tNbs0hAwVZ38LvP6_SSPelemrbG0i2uL5CxqLOJ_5A/viewform)より、Misskiraraユーザーなら誰でもカスタム絵文字の登録申請が可能です。

カスタム絵文字制作時の注意点について、いくつか記載させて頂きます。


# 1.審査について
カスタム絵文字の登録に際しましては要件や各種権利関係の確認のために、  
**モデレーターによる審査** を行っております。  

審査の結果、登録承認となったカスタム絵文字は、システムにより自動で登録処理が行われます。  
審査の結果につきましては、審査が完了し次第[@misskirara_emoji](https://misskirara.net/@misskirara_emoji) よりDMで連絡させていただきます。  
登録に至った際には、同アカウントより下記のような**新規カスタム絵文字追加のお知らせ** が行われます。  
<a href="https://objstorage.misskirara.net/register_emoji_info.png"><img src="https://objstorage.misskirara.net/register_emoji_info.png" alt="sampleimage" width="500"></a>    


審査の内容は概ね以下の通りになります。
- 画像形式（詳しくは下記）
- 画像容量（詳しくは下記）
- 絵文字の視認性、可読性
- 他者に対する権利侵害の有無
- NSFWに該当しないか
- その他[利用規約](https://doc.misskirara.net/#terms)に抵触するような内容でないか

現在のところ、既存絵文字との内容の重複に関しては却下事由とはしておりません。  
また、権利に問題のあるカスタム絵文字を発見された際は、[@Misskirara](https://misskirara.net/@MissKirara) までDM頂けますと幸いです。



# 2.画像形式について
## a.ファイル形式について
Misskiraraカスタム絵文字は投稿フォームにも記載があります通り、

<center>.png .gif .webp .avif</center>

上記4形式のみ受け付けております。（透過も機能します）  
Googleフォームの仕様上、投稿フォーム上は**全ファイル形式が選択可能な状態** になっておりますが、  
審査の段階で **却下** となりますのでご注意ください。  

## b.画像サイズについて
横：最大600px程度  
縦：最小40px程度  
リアクションで利用する想定の場合、基本的に40px前後だと過不足ないと思います。  
但し、ノート上で拡大して使用する可能性が高そうな絵文字は縦も大きめに作成するのが無難です。    
縦横比については概ね下記の比率を超えると、リアクションが画面からはみ出ますのでご注意ください。  
スマートフォン版 → 1:10  PC版 → 1:20  
  
また、上下左右の余白はそのまま絵文字に反映されます。  
基本的にはカスタム絵文字の余白は極力までカットすることをオススメします。

# 3.ファイルの容量について
現在Misskiraraにおいて、カスタム絵文字のファイル容量は  
GIF及びイラストは **24KB** 、装飾文字は**8KB**に制限されております。  
これはカスタム絵文字はタイムライン上で大量に利用されることが想定されており、  
容量の大きなカスタム絵文字が大量に投稿されることで    
ネットワークトラフィックに重大な問題が発生すること防ぐための措置となります。

**画像の容量削減に関するTIPS**
- 画像サイズを変更する  
  特にリアクションに多く用いられることが想定されるカスタム絵文字は  
  画像サイズが横400px程度でも概ね問題なく表示されます。  
  [外部の変換サイト](https://www.iloveimg.com/ja/resize-image/resize-png)によって
  画像のサイズを縮小することで容量削減が見込めます。

- ファイル形式を変更する  
  [外部の変換サイト](https://convertio.co/ja/)によって形式をwebpやavifに変換することで容量削減が可能な場合があります。

- ファイル圧縮を行う  
  png形式ファイルは色数を抑えることで、圧縮により大きくファイル容量の削減が見込めます。  
  [外部の変換サイト](https://www.iloveimg.com/ja/compress-image/compress-png)によって画像の圧縮を行うことで容量削減が見込める場合があります。


また、使用状況によってモデレーターが、カスタム絵文字の縮小、圧縮を行う場合がございます。  
この際は極力、画像の品質が損なわれないように配慮させて頂きますが、  
劣化、変質が避けられない場合がありますので、その点ご了承いただければと思います。

# 4.カスタム絵文字とMisskeyのテーマについて
カスタム絵文字の視認性、可読性には注意してください。  
Misskeyには**LightテーマとDarkテーマの2つのテーマ**があります。  
どちらのテーマでも視認性、可読性をある程度担保できる絵文字をデザインして頂けると助かります。  
現在のMisskiraraのノートの背景色は下記になります。  

<table>
<tr><td>Lightテーマ(#FFF0FB)</td><td>Darkテーマ(#1E0114)</td></tr>  
<tr><td><span style="color:#FFF0FB; background-color:#FFF0FB; font-size:180px;">　　</span></td><td><span style="color:#1E0114; background-color:#1E0114; font-size:180px;">　　</span></td></tr>
</table>

また、リアクションとして利用した際の背景色は下記になります。  
<table>
<tr><td>Lightテーマ(#F2E4EE)</td><td>Darkテーマ(#290E20)</td></tr>  
<tr><td><span style="color:#F2E4EE; background-color:#F2E4EE; font-size:180px;">　　</span></td><td><span style="color:#290E20; background-color:#290E20; font-size:180px;">　　</span></td></tr>
</table>

極端に視認性、可読性が低い絵文字につきましては、審査において却下になる可能性があります。


# 5.カスタム絵文字の差し替えについて
カスタム絵文字の差し替えについては登録用の申請フォームから行って頂くことが可能です。  
登録区分の項目で「差し替え」を選んでいただければと思います。  
但し、差し替えについては登録処理が手動で行われております関係上、  
差し替え完了時にはDM連絡が行われませんのでご了承いただければと思います。

# 6.カスタム絵文字の削除について
一度作成されたカスタム絵文字は、こちらの[削除用フォーム](https://docs.google.com/forms/d/e/1FAIpQLScRmrUXgQxuHQlFeaH9mxSPBGsHvguxbT_XHI7bSHgAYsVbsw/viewform)より削除申請が可能です。  
カスタム絵文字の削除については、基本的に当該カスタム絵文字の登録者のみが可能となっております。  
※事由によっては第三者による申請が認められる場合があります。  

# 7.カスタム絵文字の著作権等について
## 著作権について
申請頂きましたカスタム絵文字の著作権については、その制作者に著作権が帰属します。  
尚、規約に伴う下記の制限があることにご留意ください。  

## ライセンスについて
カスタム絵文字のライセンスは、申請時に下記の6種の条件についての可否が設定できます。  
- **カスタム絵文字以外の形での利用**  
  カスタム絵文字を画像として保存し、Misskirara内で添付ファイルとして利用可能か  
- **Misskirara外でのカスタム絵文字としての利用**  
  カスタム絵文字をMisskiraraからインポートすることで  
  外部のインスタンスでカスタム絵文字として利用可能か
- **Misskirara外でのカスタム絵文字以外の形での利用**  
  カスタム絵文字を画像として保存し、Misskirara外部で添付ファイルとして利用可能か
- **利用時の著作者表記省略**  
  Misskirara外での利用時に著作者表記（License欄への表記など）の省略が可能か
- **改変**  
  登録されたカスタム絵文字を2次利用する際に改変が可能か
- **商用利用**  
  登録されたカスタム絵文字を2次利用する際に商用利用が可能か

登録されたカスタム絵文字のライセンスについては下記の手順で確認を行うことができます。  
①画面左上のMisskiraraマークをクリックします。  
②「カスタム絵文字」をクリックします。  
③ライセンスを確認したい絵文字をクリックします。  
④「情報」をクリックします。

## Misskirara利用規約による上記権利の制限について
下記の場合は著作権行使、及びライセンスの適用が制限されます。

- トラフィック上の問題によりカスタム絵文字の縮小、圧縮を必要とする場合
- Misskirara画面のスクリーンショット及びノート埋め込み等の方法によってMisskirara外部に対して
  Misskirara画面を共有する際に、共有した媒体にカスタム絵文字が含まれていた場合

ライセンス条件の<b> <u>「利用時の著作者表記省略」及び「改変」以外を×</u> </b>に設定した上で、  
<b> <u> その旨及びパロディ元の作品名を「その他」の欄に記入してください。</u> </b>  
※作中セリフを装飾文字にしただけの絵文字や、  
　パロディ元が想起できない程度に改変された絵文字であれば必要はありません。
